# SpongeBob OpenGL Animation 🧽

## Description
A 2D OpenGL animation featuring SpongeBob SquarePants in a sunset oasis scene with:
- Animated waterfall
- Flying birds
- Swimming fish
- Giant tree
- Flying carpet
- SpongeBob character (moveable)
- Background music

## Controls
| Key | Action |
|-----|--------|
| W | Move up |
| S | Move down |
| A | Move left |
| D | Move right |

## How to Run

### Requirements
- MinGW-w64 (GCC 15.x)
- freeglut library
- Windows OS
